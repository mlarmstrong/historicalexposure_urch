This directory holds all R scripts used in this study.
* HistoricalExposure_counts_n_survival.R --> code for analysis of developmental success and survival count data
* HistoricalExposure_SizeData.R --> code to analyze all morphological data collected for each developmental stage
* HHistoricalExposure_Abn.R --> code to analyze all developmental abnormality data (proportion and type) for each developmental stage
* HistExposure_DEGs.R --> code using DeSeq2 to assess differential gene expression for two models (~Population + Treatment or ~urban +Treatment) in each developmental stage
* HistExposure_DEGs_interaction.R --> code using DeSeq2 to assess differential gene expression due to the interaction of urbanization and treatment in each developmental stage
* HistExposure_WGCNA.R --> code to run a weighted gene network correlation analysis (WGCNA) for each of our stages of interest to identify top modules
* HistExposure_TopGO.R --> code for analyzing Top Gene Ontology Analyses for both DEGs and WGCNA modules
*urban_sitemap.R --> code to create a site map of collected urchins
