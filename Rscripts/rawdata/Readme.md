This directory contains all raw count data for developmental success counts, survival counts, developmental abnormalities and morphological measurements, 

* HistoricalEx_Devcheck.csv --> developmental success count data
* HE_Survival-Blast.csv --> blasula count data
* HE_Survival-Gast.csv --> gastrula count data (not used in final analysis)
* HE_Survival-Plut.csv --> pluteus count data

* HE_Blastula_abn-counts.csv --> abnormality count data for the blastula stage
* HE_Blastula_abn-type.csv --> abnormality type data for the blastula stage
* HE_Gastrula_abn-counts.csv --> abnormality count data for the gastrula stage
* HE_Gastrula_abn-type.csv --> abnormality type data for the gastrula stage
* HE_Pluteus_abn-count.csv --> abnormality count data for the pluteus stage
* HE_Pluteus_abn-type.csv --> abnormality type data for the pluteus stage

* HE_Blastula_measurements.csv --> measurement data for blastula size
* HE_Gastrula_measurements.csv --> measurement data for gastrula height and gastrula stomach length
* HE_Pluteus7dpf_measurements.csv --> measurement data for pluteus body length and pluteus arm length
